# Ciberpunk Trader ⚡

## 🚀 Características
- Consulta automática de posiciones y equity de varias wallets.
- Visualización de PnL, liquidación, leverage y funding.
- Modal interactivo con historial de funding.
- Estilo cyberpunk con efectos neón.

## 📂 Estructura
- `index.html` → Página principal.
- `style.css` → Estilos neón azul/rojo.
- `dashboard.js` → Lógica para consultar el API y renderizar posiciones.
